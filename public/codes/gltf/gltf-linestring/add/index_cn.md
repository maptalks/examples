#### GLTFLineString的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| coordinates | 坐标串 | Array<Coordinate> | null           |
| options.symbol | gltf的symbol对象 | Object | null           |
| options.gapLength  | 模型外包围盒的间距     | Number  | 0             |
| options.scaleVertex  | 是否缩放端点处的模型     | Bool  | true             |
