#### InSightAnalysis 的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| invisibleColor | 通视线不可视部分的颜色 | Array | [1, 0, 0]           |
| visibleColor | 通视线可视部分的颜色 | Array | [0, 1, 0]           |
| lines  | 通视线数组      | Array  | 1             |
